# Databricks Data Engineering Portfolio

This repository contains end-to-end data engineering projects aligned with
Databricks Data Engineer Associate and Professional certifications.

The projects demonstrate batch and streaming pipelines using PySpark,
Spark SQL, Delta Lake, Structured Streaming, and Databricks Workflows.

---

## Technologies Used
- Databricks
- Apache Spark (PySpark)
- Spark SQL
- Delta Lake
- Structured Streaming

---

## Repository Structure

associate-projects/
professional-projects/
reusable-code/
interview-prep/

Each project folder contains:
- Notebooks
- SQL scripts
- Architecture diagram
- Screenshots
- README with step-by-step instructions

---

## Highlights
- Built Bronze → Silver → Gold pipelines
- Implemented MERGE, CDC, and SCD Type 2
- Developed streaming ingestion with Auto Loader
- Applied performance tuning using OPTIMIZE and ZORDER
